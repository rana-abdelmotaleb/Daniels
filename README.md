# Daniels Portfolio Website

**Daniels** is a modern and fully responsive personal portfolio website designed to showcase skills, projects, and services in a clean and professional way. Built using **HTML**, **CSS**, **Bootstrap 5**, and **FontAwesome**, it emphasizes smooth animations, responsive design, and a user-friendly interface.

- **Testimonials & Clients:** Carousel section for client feedback and endorsements.  
- **Team Section:** Display team members with social media links.  
- **Contact Form:** Simple contact form for user inquiries.  
- **Counter Section:** Shows statistics such as completed projects and happy clients.
